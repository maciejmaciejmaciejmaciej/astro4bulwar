# BulwaR Astro Site POC

This folder contains the first Astro build-time proof of concept.

It proves the final target direction:

- Astro fetches `page_builder_schema` from WordPress during build
- Astro resolves WooCommerce product sources during build
- Astro outputs static HTML pages

Current build-time route:

- `/testowa-blueprint`

Commands:

- `npm install`
- `npm run build`
- `npm run dev`
